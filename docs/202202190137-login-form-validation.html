202202190137 Валидация формы ввода Имени пользователя

Третий этап реализации формы входа, второго модуля по WorldSkills КОД 1.3.

  1. В последних версиях HTML5 у input-ов появилось много полезных атрибутов для валидирования формы на стороне клиента:
    • Чтобы пользователь не указал пустое имя, нам поможет атрибут required - говорящий браузеру, что данное поле обязательно для заполнения
    • Чтобы пользователь вводил только буквы/цифры и не более 20-ти, мы можем воспользоваться атрибутом pattern, указывающим регулярное выражение, которому должно соответствовать введенное значение
    • Чтобы при валидации возникала понятное пользователю описание, что нужно ввести в это поле нам поможет атрибут title.
